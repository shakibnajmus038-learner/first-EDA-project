# 📊 Netflix EDA Project

This is an Exploratory Data Analysis (EDA) project on the Netflix dataset using Python and pandas. The goal is to analyze Netflix content data to uncover insights about movies and TV shows available on the platform.

---

## 📁 Dataset

The dataset used is: ⁠ Netflix Dataset.csv ⁠  
It includes columns like:
•⁠  ⁠Title
•⁠  ⁠Type (Movie/TV Show)
•⁠  ⁠Director
•⁠  ⁠Cast
•⁠  ⁠Country
•⁠  ⁠Date Added
•⁠  ⁠Release Year
•⁠  ⁠Rating
•⁠  ⁠Duration
•⁠  ⁠Genre

## 📈 Key Insights

Some of the questions I answered in this project:

•⁠  ⁠How many movies vs TV shows are on Netflix?
•⁠  ⁠What’s the trend of content added each year?
•⁠  ⁠Which countries produce the most Netflix content?
•⁠  ⁠Who are the most frequent directors?
•⁠  ⁠What are the most common genres?
•⁠  ⁠How long are the most typical movies and shows?
